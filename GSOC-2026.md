## Summary

This project focuses on building developer tooling to reduce configuration errors. The work includes implementing the LSP specification to provide real-time syntax highlighting, code completion, and semantic diagnostics (e.g., detecting hardware mismatches), packaged within a VS Code extension.

# Implement Advanced IR Optimization Passes

## Summary
Harden the compiler by implementing standard optimization algorithms within the Intermediate Representation (IR).

This project focuses on improving the efficiency of the generated binary code. The work involves implementing graph-based algorithms for Dead Code Elimination (DCE), Common Subexpression Elimination (CSE), and Constant Folding to minimize model size and execution overhead.

- https://en.wikipedia.org/wiki/Control-flow_graph
